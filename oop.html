Programación Orientada a Objetos (POO)

La POO se basa en el uso de objetos y clases para organizar el código.

Conceptos Clave

  • Clases: Plantillas para crear objetos.
  • Objetos: Instancias de una clase.
  • Herencia: Permite crear nuevas clases a partir de otras.
  • Encapsulación: Oculta los detalles internos de una clase.
  • Polimorfismo: Permite que un objeto se comporte de múltiples formas.

Ejemplo Práctico


// Ejemplo en JavaScript
class Animal {
  constructor(nombre) {
    this.nombre = nombre;
  }

  hacerSonido() {
    console.log("Sonido genérico");
  }
}

class Perro extends Animal {
  hacerSonido() {
    console.log("Guau guau");
  }
}

const miPerro = new Perro("Rex");
miPerro.hacerSonido(); // Salida: Guau guau
